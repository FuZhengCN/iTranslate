隐私政策

最后更新：2026-05-23

本隐私政策适用于 通译（iTranslate） 浏览器扩展（以下简称"本扩展"）。我们高度重视您的隐私，特此说明本扩展如何处理您的数据。

1. 数据收集与存储

本扩展收集和存储以下数据：

  • API 配置信息（API 端点地址、API 密钥、模型名称）：存储在浏览器本地 chrome.storage.sync 中，仅用于调用您指定的 AI 翻译服务。
  • 翻译缓存（原文 + 译文）：存储在浏览器本地 IndexedDB 中，用于避免重复翻译相同内容，减少 API 消耗。
  • 用户语言偏好（源语言、目标语言选择）：存储在浏览器本地 chrome.storage.sync 中，用于记住您的翻译偏好。

所有数据均存储在您的浏览器本地。本扩展开发者不拥有任何服务器，不收集、不上传、不存储您的任何个人数据或翻译内容。

2. 数据传输

当您触发翻译功能时，网页中的文本内容会被发送到您自行配置的第三方 AI API 服务（如 DeepSeek、OpenAI 等）以获取翻译结果。本扩展开发者不参与、不中转、不记录此数据传输过程。您对所使用的第三方 API 服务的隐私政策承担了解责任。

3. 页面访问权限

本扩展请求以下权限：

  • activeTab：仅当您主动点击扩展图标触发翻译时，才能访问当前标签页的网页内容，不会在后台静默读取页面。
  • storage：用于在浏览器本地存储 API 配置和翻译缓存。
  • scripting：用于向网页注入翻译功能所需的脚本和样式。
  • 主机权限（http://*/*、https://*/*）：使翻译功能可在任意网页上运行。

4. 第三方服务

本扩展依赖您配置的第三方 AI API 服务完成翻译。本扩展本身不包含任何第三方分析、广告或跟踪代码。除了您指定的 API 端点外，不会向任何其他服务器发送请求。

5. 数据安全

您的 API 密钥存储在浏览器本地存储中，不会离开您的设备（除发送至您指定的 API 服务进行身份验证外）。建议您妥善保管 API 密钥，不要与他人共享。

6. 用户权利

您可以随时：

  • 在扩展设置中修改或删除 API 配置
  • 通过"清除缓存"按钮删除所有本地翻译缓存
  • 卸载扩展以完全清除所有本地存储的数据

7. 政策更新

本隐私政策可能不时更新。更新后的政策将随扩展新版本发布。继续使用本扩展即表示您同意更新后的隐私政策。

8. 联系方式

如有隐私相关问题，请通过以下渠道联系：

  • GitHub Issues：github.com/FuZhengCN/iTranslate/issues
  • Gitee Issues：gitee.com/fuzheng0312/i-translate/issues

Privacy Policy

Last updated: May 23, 2026

This Privacy Policy applies to the iTranslate (通译) browser extension (hereinafter "the Extension"). We take your privacy seriously and explain here how the Extension handles your data.

1. Data Collection & Storage

The Extension collects and stores the following data:

  • API Configuration (endpoint URL, API key, model name): stored locally in chrome.storage.sync, used solely to call your designated AI translation service.
  • Translation Cache (source text + translated text): stored locally in IndexedDB to avoid re-translating identical content and reduce API consumption.
  • Language Preferences (source language, target language): stored locally in chrome.storage.sync to remember your translation preferences.

All data is stored locally in your browser. The Extension developer does not own any servers and does not collect, upload, or store any of your personal data or translation content.

2. Data Transmission

When you trigger translation, text content from the webpage is sent to the third-party AI API service you have configured (such as DeepSeek, OpenAI, etc.) to obtain translation results. The Extension developer does not participate in, relay, or log this data transmission. You are responsible for understanding the privacy policies of the third-party API services you use.

3. Page Access Permissions

The Extension requests the following permissions:

  • activeTab: Accesses the current tab's content only when you actively click the extension icon to trigger translation. Does not silently read pages in the background.
  • storage: Used to store API configuration and translation cache locally in the browser.
  • scripting: Used to inject translation scripts and styles into web pages.
  • Host permissions (http://*/*, https://*/*): Enables the translation feature to work on any web page.

4. Third-Party Services

The Extension relies on the third-party AI API service you configure to perform translations. The Extension itself contains no third-party analytics, advertising, or tracking code. No requests are sent to any server other than the API endpoint you specify.

5. Data Security

Your API key is stored in your browser's local storage and does not leave your device (except when sent to your designated API service for authentication). We recommend keeping your API key secure and not sharing it with others.

6. User Rights

You may at any time:

  • Modify or delete API configuration in the extension settings
  • Delete all local translation cache via the "Clear Cache" button
  • Uninstall the Extension to completely remove all locally stored data

7. Policy Updates

This Privacy Policy may be updated from time to time. Updated policies will be published with new versions of the Extension. Continued use of the Extension constitutes acceptance of the updated Privacy Policy.

8. Contact

For privacy-related inquiries, please contact us via:

  • GitHub Issues: github.com/FuZhengCN/iTranslate/issues
  • Gitee Issues: gitee.com/fuzheng0312/i-translate/issues
