# Chrome Web Store 截图指南

## 准备

1. Chrome 加载 `dist/` 目录（已构建）
2. 在设置页配置好 API Key（测试连接通过）
3. 打开 `docs/store/test-page.html`（或任意英文网页）

## 需要截取的 5 张图（1280×800 或 640×400）

### 1. Popup 弹窗界面
- 点击工具栏通译图标
- 截取弹出的面板（含语言选择、翻译/撤销按钮、划词翻译开关）
- 建议：640×400 即可，突出弹窗内容

### 2. 设置页面
- 点击弹窗中的齿轮图标进入设置页
- 截取完整设置页（API Endpoint / Key / Model / System Prompt）
- 注意遮挡 API Key 敏感部分

### 3. 翻译效果（核心截图）
- 打开 `docs/store/test-page.html`
- 点击「翻译此页面」
- 等待翻译完成
- 截取译文嵌入原文下方的双语对照效果

### 4. 划词翻译
- 在弹窗中开启「划词翻译」开关
- 在页面中选中一段英文文字
- 等待翻译气泡弹出
- 截取气泡 + 选中文字

### 5. 工具栏图标
- 截取浏览器工具栏区域
- 展示通译图标在扩展栏中的位置

## 截图格式要求

- PNG 或 JPEG
- 分辨率：1280×800 或 640×400
- 去除浏览器书签栏等无关元素
- 不要包含个人敏感信息

## 命名

```
01-popup.png       — 弹窗界面
02-settings.png    — 设置页面
03-translation.png — 翻译效果对比
04-selection.png   — 划词翻译气泡
05-toolbar.png     — 工具栏图标
```

保存到 `docs/store/screenshots/`。
