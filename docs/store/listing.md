# Chrome Web Store 上架文案

## 简短描述（≤132 字符）

> AI 双语对照翻译。译文嵌入原文下方，不破坏布局。需自备 AI API Key（DeepSeek/OpenAI 等），数据全本地。

---

## 详细描述

> **使用本扩展需要提供你自己的 AI API 密钥（Key）。这不是免费翻译工具，不会为你提供翻译服务。如果你不熟悉"API Key"是什么，请在安装前先了解清楚，以免差评。**

**通译（iTranslate）让你直接阅读任何语言的网页，就像读母语一样自然。**

### 为什么你需要通译

每天，互联网上产生的内容超过 70% 不是中文。技术文档、学术论文、行业资讯、产品评测——最优质的信息往往是用英文写成的。传统的做法是在翻译工具和原文页面之间来回切换，既打断阅读节奏，又丢失上下文。

通译的做法不同：它把 AI 翻译结果**直接嵌入到原文下方**，用略浅的文字呈现。你不需要跳转到任何其他页面，也不需要把整个网页替换成译文。原文和译文同时存在、上下对照——读得懂的地方看原文，读不懂的地方扫一眼下方译文，一气呵成。

### 它和普通翻译扩展有什么不同

大多数网页翻译扩展做的是"替换"——把整页文字换成译文，原文就消失了。这带来两个问题：一是翻译错误时你无法对照原文纠错；二是页面布局经常被撑破变形。

通译做的是"叠加"——原文保留不动，译文作为附加层插入到每个段落下方。页面结构、图片、代码块、排版完全不受影响。你甚至可以把译文当作"隐形字幕"，只在需要时视线下移查看。

### 核心优势

**AI 大模型驱动，不是机翻。** 通译调用的是 GPT、DeepSeek、Claude 等大语言模型，不是传统的统计机器翻译。它能理解上下文、识别专业术语、处理长难句——翻译质量远超浏览器自带的网页翻译。

**你用你自己的 API Key。** 通译不提供翻译服务，它只是一个连接你和 AI 模型的桥梁。你使用自己的 API 密钥，数据直接从你的浏览器发送到你指定的 AI 服务。没有任何中间服务器经手你的数据。

**所有数据本地存储。** API 密钥、翻译缓存、语言偏好——全部存在你的浏览器里。不注册账号、不上传数据、不植入广告和跟踪代码。你的浏览隐私和翻译内容完全由你掌控。

**翻译缓存帮你省钱。** 翻译过的句子自动缓存到本地，下次遇到相同内容直接秒出结果，不消耗 API 额度。浏览同一网站的多篇文章时，缓存命中率很高。

### 功能一览

- 一键翻译整页，译文以浅色文本插入原文下方
- 选中任意文字，弹窗即时翻译
- 支持中文、英文、日文、韩文、法文、德文互译
- 自动检测页面语言，智能匹配目标语言
- 自定义 API 端点、模型名称和翻译提示词
- 一键撤销翻译，页面恢复原样
- 极地冰川主题浅色 UI，视觉舒适不刺眼

### 适用人群

- **开发者 & 工程师** — 阅读英文技术文档、GitHub Issues、Stack Overflow
- **科研 & 学术** — 浏览外文论文、期刊摘要、学术会议网站
- **产品 & 运营** — 调研海外竞品、阅读行业分析报告
- **跨境电商** — 查看海外商品描述、买家评价
- **语言学习者** — 双语对照阅读，随时对照原文和译文

### 开始使用

1. 安装扩展后，准备好你的 AI API 密钥（支持 DeepSeek、OpenAI、Claude 等兼容接口）
2. 点击扩展图标 → 齿轮 → 填入 API 信息 → 测试连接 → 保存
3. 打开任意网页，点击扩展图标，选择语言，点击"翻译此页面"

AI API 密钥获取渠道：DeepSeek（platform.deepseek.com）、OpenAI（platform.openai.com），通常注册即送免费额度。

## 权限说明

| 权限 | 为什么需要 |
|------|-----------|
| `activeTab` | 只有你点击扩展图标时，才能访问当前页面内容。不会在后台偷偷读你的网页 |
| `storage` | 把你的 API 密钥、翻译缓存、语言偏好存在浏览器本地 |
| `scripting` | 把翻译文字渲染到网页上 |
| 访问所有 HTTP/HTTPS 网站 | 让你在任何网页上都能使用翻译功能 |

**通译不会：** 收集你的浏览历史、上传你的数据到开发者服务器、植入广告或跟踪代码。开发者根本就没有服务器。

## 隐私

所有数据（API 密钥、翻译记录、个人偏好）均存储在你的浏览器本地。翻译文本直接发送到你指定的 AI 服务，不经任何中转。详见[隐私政策](https://fuzhengcn.github.io/iTranslate/privacy)。
