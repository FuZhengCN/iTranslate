## 每章固定结构

1. 本章目标（3 句话）
2. 实战任务（真实代码 + CC 对话记录）
3. 核心技巧（本章学到的 Prompt/策略）
4. 小结（卡片式要点）

## 内容风格

- 语言：简体中文，口语化但专业——像资深同事在讲
- 代码：关键字/标识符保留英文，注释用中文
- 对话记录：精选关键回合 + 「为什么这样问」标注框分析
- 篇幅：每章 3000-5000 字，全书 7-10 万字
